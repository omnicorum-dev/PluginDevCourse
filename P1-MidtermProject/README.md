# Project 1 - Midterm Project

Build your own small synthesizer with what you have learned!

Please attempt to build this plugin ON YOUR OWN!
Put together all the classes we have built thus far!
You shouldn't need to construct much new material,
but just put together the pieces in creative ways.
Have fun with it!

My own implementation will be shown in the Project 1 Episode.

Things to consider adding:

- Polyphony (8 voices)
- Envelopes
- Filters
- Delay
- Distortion
- Anything else fun you can think of!

Challenge: Add unison!

- Multiple voices per note
- Each are slightly detuned from center
- Each are panned slightly away from center
