# Project 2 - Final Project

Build a resynthesis engine plugin with what you have learned!

A resynthesis engine is an EFFECT that takes an input signal
and *resynthesizes* the most prominent few frequencies with a
specified oscillator shape.

Like the midterm project, please attempt to build this plugin
ON YOUR OWN, starting with this description and seeing how far you can get.
Break down the problem and try and see how you can use
the pieces we have built thus far to accomplish the task.

My own implementation will be shown in the Project 2 Episode.
